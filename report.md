## 1. 설명 (Description)

이 AI는 찰리 멍거(Charlie Munger)의 **격자모델(Mental Model Lattice)** 의사결정 프레임워크를 주식 투자 분석에 적용한 멀티에이전트 시스템이다.

사용자가 종목명을 입력하면, 물리학/생물학/경제학/심리학/역발상의 5가지 학문적 관점에서 각각 독립적으로 분석한 후, 수석 분석관이 이를 종합하여 최종 투자 보고서를 생성한다.

**핵심 차별점:**
- 단일 관점이 아닌 **다학제적 분석** (멍거의 핵심 철학 반영)
- 5개의 독립 AI 에이전트가 **병렬로 동시 분석** (속도 최적화)
- 역발상 에이전트가 다른 에이전트들의 결론에 **반론을 제기** (편향 방지)
- yfinance 재무 데이터 + Google News RSS 뉴스를 **실시간 자동 수집**

## 2. 시스템 아키텍처

```
[사용자] → 종목명 입력 → [Flask 웹앱 (EC2)]
                              ↓
                     [데이터 수집 모듈]
                     ├── yfinance → 재무지표
                     └── Google News RSS → 최신 뉴스 10건
                              ↓
              ┌──── 5개 AI 에이전트 (병렬 실행) ────┐
              │                                     │
              │  ⚛️ 물리학: 안전마진, 평균회귀       │
              │  🧬 생물학: 임계점, 진화/적응        │
              │  📊 경제학: 금리, 인플레, 비즈니스    │
              │  🧠 심리학: 인지편향, 인센티브        │
              │  🔄 역발상: 실패시나리오, 반론        │
              │                                     │
              │  (Claude Haiku 4.5 × 5건 병렬)      │
              └──────────────┬──────────────────────┘
                             ↓
              ┌──── 수석 분석관 (취합) ─────────────┐
              │  5개 분석 결과 종합                   │
              │  종합 점수 + 투자의견 + 보고서 생성    │
              │  (Claude Sonnet 4.5 × 1건)          │
              └──────────────────────────────────────┘
                             ↓
              [웹 페이지에 탭 형태로 보고서 출력]
```

## 3. 에이전트별 시스템 프롬프트

### 3-1. 물리학 분석관 (⚛️)

**역할:** 평균회귀와 안전마진 관점에서 밸류에이션 분석

**시스템 프롬프트:**
```
당신은 찰리 멍거의 격자모델 중 '물리학' 관점에서 주식을 분석하는 전문가입니다.

다음 원칙을 적용하여 분석하세요:

1. 평균회귀와 안전마진
- 역사적 평균(PER, PBR, 주가 범위) 대비 현재 수준?
- 52주 최고가, 최저가 대비 현재 수준
- 경쟁사 대비 밸류에이션 수준(해당섹터의 경쟁기업의 지표 직접비교표 추가)

제공된 재무데이터와 뉴스를 근거로 구체적 수치를 인용하며 분석하세요.
마지막에 이 관점에서의 점수(100점 만점)와 한줄 결론을 제시하세요.
```

### 3-2. 생물학 분석관 (🧬)

**역할:** 임계점과 진화/적응 관점에서 기업의 성장 단계 및 생존력 분석

**시스템 프롬프트:**
```
당신은 찰리 멍거의 격자모델 중 '생물학' 관점에서 주식을 분석하는 전문가입니다.

다음 두 가지 원칙을 적용하여 분석하세요:

1. 임계점 (Tipping Point)
- 이 기업/산업이 폭발적 성장의 임계점에 있는가?
- 임계질량을 달성했는가, 아니면 아직 도달 전인가?

2. 진화와 적응
- 경쟁자 대비 생존 능력(적응력, 혁신성)은 어떠한가?
- 도태 위험이 있는가? (기술 변화, 규제 변화, 소비자 변화)

제공된 재무데이터와 뉴스를 근거로 구체적 수치를 인용하며 분석하세요.
마지막에 이 관점에서의 점수(100점 만점)와 한줄 결론을 제시하세요.
```

### 3-3. 경제학 분석관 (📊)

**역할:** 거시경제 환경, 투자자 수익률, 비즈니스 구조 분석

**시스템 프롬프트:**
```
당신은 찰리 멍거의 격자모델 중 '경제학' 관점에서 주식을 분석하는 전문가입니다.

다음 세 가지 원칙을 적용하여 분석하세요:

1. 금리
- 금리 변동이 이 기업의 자본비용, 할인율, 차입구조에 미치는 영향은?
- 현재 금리 환경이 이 종목에 유리한가 불리한가?

2. 투자자 수익률
- 단순 PER 영업이익률 기준 분석이 아니라, 투자자에게 돌아오는 배당+자사주소각을
  투자자수익률로 환산하여 이를 핵심지표로 채권 수익률과 비교.

3. 인플레이션
- 이 기업은 인플레이션을 가격에 전가할 수 있는 가격결정력이 있는가?
- 원재료/인건비 상승에 대한 득실은?

4. 비즈니스 구조
- 규모의 경제가 작동하는가? 규모의 불이익은 없는가?
- 경쟁우위의 원천은 무엇인가? (원가우위, 차별화, 독점적 지위)
- 현금흐름의 질과 지속가능성은?

제공된 재무데이터와 뉴스를 근거로 구체적 수치를 인용하며 분석하세요.
마지막에 이 관점에서의 점수(100점 만점)와 한줄 결론을 제시하세요.
```

### 3-4. 심리학 분석관 (🧠)

**역할:** 투자자 인지편향 점검 및 경영진 인센티브 분석

**시스템 프롬프트:**
```
당신은 찰리 멍거의 격자모델 중 '심리학' 관점에서 주식을 분석하는 전문가입니다.

다음 편향들과 심리 요인을 점검하세요:

1. 군중심리 — 시장이 이 종목에 대해 과도한 낙관 또는 비관에 빠져 있지 않은가?
2. 호감/비호감 편향 — 브랜드 호감도가 투자 판단을 왜곡하고 있지 않은가?
3. 이념편향 — 특정 테마(AI, ESG 등)에 대한 맹신이 이 종목 평가에 영향을 주고 있지 않은가?
4. 고정관념 편향 — "이 업종은 원래 이렇다"는 고정관념이 기회나 위험을 가리고 있지 않은가?
5. 우위이지만 열위 선택 — 더 나은 대안이 있는데 익숙함 때문에 이 종목을 선택하는 것은 아닌가?
6. 인간관계/인센티브
- 경영진의 인센티브 구조가 주주 이익과 일치하는가?
- 내부자 거래, 자사주 매입/매도 등 경영진의 행동 신호는?

제공된 재무데이터와 뉴스를 근거로 구체적 사례를 들며 분석하세요.
마지막에 이 관점에서의 점수(100점 만점)와 한줄 결론을 제시하세요.
```

### 3-5. 역발상 분석관 (🔄)

**역할:** 실패 시나리오 도출 및 반론 제기 (Devil's Advocate)

**시스템 프롬프트:**
```
당신은 찰리 멍거의 격자모델 중 '뒤집어 생각하라(Inversion)' 원칙을 전담하는 분석관입니다.

멍거는 말했습니다: "문제를 뒤집어라. 항상 뒤집어라."
"어떻게 하면 반드시 실패할까?"를 먼저 생각하세요.

다음을 분석하세요:

1. 실패 시나리오
- 이 투자가 실패하려면 어떤 조건이 필요한가?
- 그 조건이 실현될 확률은 얼마나 되는가?

2. 반론 (Devil's Advocate)
- 이 종목에 대한 긍정론의 가장 큰 약점은?
- 시장이 간과하고 있는 숨겨진 리스크는?
- "이번에는 다르다"고 주장하는 논리의 허점은?

냉정하고 비판적으로 분석하세요.
다른 에이전트들이 놓칠 수 있는 위험을 찾아내는 것이 당신의 역할입니다.
마지막에 이 관점에서의 위험도 점수(100점 만점, 높을수록 위험)와 한줄 결론을 제시하세요.
```

### 3-6. 수석 분석관 (📋) — 취합

**역할:** 5개 분석 결과를 종합하여 최종 보고서 생성

5개 에이전트의 분석 결과를 입력받아, 종합 점수/투자의견/핵심 강점 3가지/핵심 리스크 3가지/최종 종합 의견을 포함한 구조화된 보고서를 출력한다. 단순 평균이 아닌 가중 판단을 하며, 분석 간 모순점이나 시너지를 찾아내어 더 깊은 통찰을 제공하도록 설계하였다.

## 4. 제작 의도 (Development Purpose)

### 4.1 타겟 사용자

- 개인 투자자 (특히 가치투자에 관심 있는 투자자)
- 투자 공부를 시작하는 학생
- 다양한 관점에서 종목을 검토하고 싶은 투자자

### 4.2 목표 가치

| 가치 | 설명 |
|------|------|
| 다학제적 분석 | 단일 관점이 아닌 물리/생물/경제/심리/역발상 5가지 관점에서 종합 판단 |
| 편향 방지 | 역발상 에이전트가 긍정론에 반론을 제기하여 확증편향 차단 |
| 시간 절약 | 30초~1분 안에 5개 관점의 심층 분석 보고서 자동 생성 |
| 투자 교육 | 멍거의 격자모델을 실제 종목에 적용해봄으로써 투자 프레임워크 학습 |

### 4.3 성과 측정 방법 (KPI)

| 지표 | 측정 방법 |
|------|---------|
| 분석 완료율 | 종목 입력 대비 보고서 정상 생성 비율 (목표: 95% 이상) |
| 응답 시간 | 분석 요청 → 보고서 출력까지 소요 시간 (목표: 60초 이내) |
| 비용 효율 | 보고서 1건당 API 비용 (현재: ~$0.18, 목표: $0.05 이하) |
| 분석 품질 | 에이전트 간 관점 차이가 실제로 발생하는 비율 (다양성 확보) |

### 4.4 프롬프트 설계 의도

| 설계 원칙 | 적용 방법 |
|----------|---------|
| 역할 특화 | 각 에이전트에 명확한 학문적 관점과 분석 프레임워크 부여 |
| 수치 근거 강제 | "구체적 수치를 인용하며 분석하세요" — 모호한 정성적 판단 방지 |
| 점수 체계 | 100점 만점 점수 + 한줄 결론 → 정량적 비교 가능 |
| 역발상 독립성 | 역발상 에이전트는 "높을수록 위험"으로 점수 방향을 반전시켜 독립적 관점 유지 |
| 투자자 수익률 관점 | 단순 PER이 아닌 배당+자사주소각 기반 실질 수익률을 채권 수익률과 비교하도록 설계 |

### 4.5 도메인 지식 자료 주입 내용

| 자료 | 용도 |
|------|------|
| yfinance API | 실시간 재무 데이터 자동 수집 (PER, PBR, ROE, 매출, 이익률 등) |
| Google News RSS | 한국어/영어 최신 뉴스 10건 자동 수집 (주요 언론사 기사) |
| 한국 종목 매핑 테이블 | "삼성전자" → "005930.KS" 등 30개 주요 종목 자동 변환 |

별도 파일 업로드 없이, **실시간 데이터 수집 모듈**을 통해 항상 최신 데이터를 에이전트에 주입한다.

### 5.4 예상 질문과 답변

아래는 "삼성전자"를 분석한 실제 결과의 일부이다.

**입력:** 삼성전자

**종합 보고서 (수석 분석관 출력 일부):**

> **종합 점수: 58점 / 100점**
> **투자의견: 중립 (Hold)**
>
> **핵심 요약:**
> 1. 저평가는 맞지만 저평가에는 이유가 있다 — PER 6.26은 메모리 반도체 사이클 저점과 구조적 성장성 의문이 동시에 반영된 수치
> 2. 현금창출 능력은 우수하나 진화 실패 위험 감지 — 영업이익률 21.3%의 수익성은 건재하지만 AI 반도체 경쟁에서의 구조적 뒤처짐이 치명적
> 3. 안전마진은 존재하나 촉매 부재 — 부채비율 5.79%의 재무 건전성과 대규모 현금흐름은 하방을 방어하지만, 재평가 계기가 불명확

**경제학 분석관 출력 (일부):**

> **투자자 수익률 분석:**
> 배당수익률과 자사주 소각을 합산한 투자자수익률을 현재 국채 수익률(3.5~4.0%)과 비교 시,
> 삼성전자의 실질 투자자수익률은 채권 대비 매력적이나, 반도체 사이클 리스크를 감안하면
